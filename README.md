# 📊 Sales Performance Dashboard | Power BI

An interactive **Power BI Sales Performance Dashboard** built to analyze sales performance, profitability, and business trends through dynamic visualizations and KPI reporting.

## 🚀 Project Overview
This project consists of **two interactive report pages** that provide insights into:
- Sales Performance
- Customer & Product Analysis
- Regional Sales Analysis
- Profitability Metrics

## 📈 Key Features
- KPI Cards: **Total Sales, Total Profit, Total Cost, Profit Margin**
- Time Intelligence: **YTD, MTD, QTD, Last Year (LY)**
- Interactive **Slicers,and Cross-Filtering**
- Customer, Product, and County-wise Sales Analysis
- Star Schema Data Model
- DAX Measures & Power Query Transformations

## 🛠️ Tools & Technologies
- Power BI Desktop
- DAX
- Power Query (M)
- Star Schema
- Data Modeling
